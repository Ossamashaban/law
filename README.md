# 🏛️ موقع المستشار إسلام الغرياني - محامي متخصص

موقع احترافي وحديث للمستشار **إسلام الغرياني** - محامي متخصص في القانون المدني والإداري

## ✨ المميزات الرئيسية

### 🎨 تصميم عصري واحترافي
- واجهة مستخدم حديثة وجذابة
- ألوان احترافية تعكس الثقة والكفاءة
- تصميم متجاوب يعمل على جميع الأجهزة

### 🎬 تأثيرات 3D متقدمة
- **مكعب 3D دوار** في قسم البطل (Hero Section)
- **بطاقات خدمات ثلاثية الأبعاد** مع تأثير الدوران عند التمرير
- **تأثيرات الزجاج المتجمد** (Glassmorphism)
- **رسوم متحركة ديناميكية** في جميع الأقسام

### 📱 تجربة المستخدم
- شريط تنقل لزج (Sticky Navigation)
- تمرير سلس بين الأقسام
- تأثيرات ظهور العناصر عند التمرير
- نماذج تواصل تفاعلية

## 🔧 تخصيص الموقع

### تغيير معلومات المستشار
عدّل النصوص في `index.html`:
```html
<h1>المستشار <span>إسلام الغرياني</span></h1>
<p class="subtitle">محامي متخصص في القانون المدني والإداري</p>
```

### تحديث بيانات الاتصال
عدّل قسم الاتصال:
```html
<p>+966 XX XXX XXXX</p>
<p>info@ghariany.law</p>
```

### إضافة صور
يمكنك إضافة صور للمستشار بإنشاء مجلد `images/` وتحديث `index.html`.
